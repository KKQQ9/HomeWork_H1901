JQ的对象、获取、报错、事件、页面载入

一、获取元素上不同

1.dom元素: document.     
exp：document.getElementById()
2.jq对象：$(dom元素)  
exp：$('#id') $('.class') $('tag')

二、dom元素与jq对象相互转换

1.dom转jq     
exp：

HtmlCode: < p class="p1" id="p1">段落< /p>

var p1Dom = document.getElementById('p1')
console.log(p1Dom)
console.log('jq', $(p1Dom))
2.jq转dom  
exp：
var p1Jq = $('p')
console.log('dom:' , p1Jq[0])
console.log(p1Dom === p1Jq[0])

三、报错机制不同

innerHTML 是js里面dom对象的属性
html() 是jquery里面jq对象的方法
1.js报错     
exp：
console.log(p1Jq.html())
console.log(p1Dom.html()) // 报错
alert(1) // js 上面报错下面不执行
2.jq不报错  
exp：
console.log(p1Dom.innerHTML)
console.log(p1Jq.innerHTML) // 不报错
alert(1) // jq 上面报错下面依旧执行

四、事件不同

HtmlCode:
< a href="" id="js">js< /a>
< a href="" id="jq">jq< /a>
1.js：on+方法名
domcument.onclick = function() {...}
exp：
var js = document.getElementById('js');
js.onclick = function() {
    alert('this is js')
}
2.jq：方法名()
$('').click( function() {} )
exp：
var jq = $('#jq');
jq.click(function(){
    alert('this is jq')
})

五、页面载入不同

1.jq加载速度比js快
domcument.onclick = function() {...}
exp：
window.onload = function(){
    console.log('选择1');
}
$(document).ready(function(){
    console.log('选择2');
})
$(function(){
    console.log('我要开始装逼了')
})
// 选择2
// 我要开始装逼了
// 选择1
2.js存在覆盖,jq不存在覆盖
